# gmm-codex-skills

这是我平时使用 Codex 安装的一些 skills，其中有一些是我自己创建的 skills，有一些是引用了一些著名的 skills，目的主要是为了方便 Codex 使用。后续如果还增加 skills 的话，也会继续推送到这里。这个仓库主要算是我个人使用的 skills 备份。

## Skills 介绍

### Core

- `skill-creator`【引用/整理】：创建或更新 Codex skills。
- `find-skills`【自建/个人定制】：查找和发现适合某类任务的 skills。
- `evaluate-suggestions`【自建/个人定制】：在采纳用户建议或修改方向前，先进行判断和权衡。

### Writing & Research

- `content-research-writer`【自建/个人定制】：支持带研究和引用的内容写作。
- `nature-academic-search`【自建/个人定制】：学术文献检索、引用验证和多来源搜索工作流。
- `paper-write`【自建/个人定制】：本科、硕士论文写作、润色、扩写、翻译和防 AIGC 等工作流。
- `patent-write`【自建/个人定制】：中文发明专利撰写、改写、统稿和权利要求处理。

### Documents & Data

- `docx`【引用/整理】：Word 文档创建、读取、编辑和格式处理。
- `pdf`【引用/整理】：PDF 读取、拆分、合并、表单填写、OCR 和格式处理。
- `xlsx`【引用/整理】：电子表格创建、清洗、编辑、公式、格式和转换。
- `excel-analysis`【自建/个人定制】：Excel 数据分析、透视表、图表和统计处理。
- `wechat-daily-report`【自建/个人定制】：分析导出的微信聊天记录，并生成日报。

### Productivity

- `file-organizer`【自建/个人定制】：文件整理、归类、去重和清理工作流。

### Dev Tools

- `mcp-builder`【引用/整理】：MCP Server 构建和工具集成指南。
- `codex-reconnecting-solve`【自建/个人定制】：排查和修复 Windows 上 Codex Desktop 反复重连的问题。
- `github-repo-publisher`【自建】：在明确收到指令后，将本地项目创建为 GitHub 仓库并推送。

### Career

- `tailored-resume-generator`【自建/个人定制】：根据岗位描述和个人背景生成定制化简历。
