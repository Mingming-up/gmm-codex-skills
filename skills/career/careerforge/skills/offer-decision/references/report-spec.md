# HTML 决策报告设计规范

> 用户同意生成 HTML 决策报告时必须读取本文件，获取完整的设计规范和 7 个报告模块的布局要求。

## 设计规范

**与 CareerForge 其他报告统一设计语言：**

- 配色系统：`--ink: #1a1a2e`、`--accent: #e94560`、`--green: #2ecc71`、`--blue: #3498db`、`--amber: #f39c12`、`--red: #e74c3c`
- 字体：Google Fonts `Noto Sans SC`（正文）+ `Noto Serif SC`（标题）
- 卡片布局：圆角 `12px`、阴影 `0 2px 12px rgba(0,0,0,.08)`
- 响应式设计：移动端友好

## 报告包含模块

1. **顶部 Header**：用户姓名、报告日期、offer 数量
2. **总览对比表**：关键数据并排对比
3. **六维度雷达图**：SVG 绘制，每个 offer 一个颜色
4. **维度详情卡片**：6 张卡片，每张展示该维度各 offer 的评分和理由
5. **经济计算明细**：年包、税后、时薪的计算过程
6. **推荐结论**：大号文字 + 核心原因 + 注意事项
7. **薪资谈判卡片**（如适用）：锚点/底线 + 话术脚本

## 文件命名

`{用户姓名}-Offer决策报告.html`
